## 多项式理论

多项式理论的本质是 “将数域上的运算推广到多项式集合，再用多项式性质分析矩阵”

> **域 → 多项式环 → 整除结构 → 因式分解 → 最小多项式 → 矩阵结构理论→ 矩阵在相似结构下的分类（有理标准型与Jordan标准型）→ 多项式的应用（插值与逼近等）**

### 多项式定义

设 n 是**非负整数**（即 \(0,1,2,3,$\dots$\)），\(a_0, a_1, a_2, \dots, a_n\) 是**常数**（称为 “系数”，可属于实数、复数或其他数域），x 是**变量**（或 “不定元”），则形如：$\boxed{f(x) = a_nx^n + a_{n-1}x^{n-1} + \dots + a_1x + a_0}$ 的表达式称为**一元多项式**（仅含一个变量 x）。若含多个变量（如 \(x, y, z\)），则称为**多元多项式**。

**零多项式，n次多项式**

规定：零多项式次数为-∞。

### 环

**定义：**一个非空集合有两个代数运算，加法，乘法，并且满足一下运算法则。

 加法需要满足交换群的所有公理，这是环的 “基础结构”：

- **加法封闭性**：对任意\(a, b \in R\)，\(a + b \in R\)（运算结果仍在集合内）；
- **加法结合律**：对任意\(a, b, c \in R\)，\((a + b) + c = a + (b + c)\)；
- **加法单位元（零元）存在**：存在唯一元素\(0 \in R\)，对任意\(a \in R\)，\(a + 0 = 0 + a = a\)（记为 “0”，是环的核心基准元之一）；
- **加法逆元存在**：对任意\(a \in R\)，存在唯一元素\(-a \in R\)，使\(a + (-a) = (-a) + a = 0\)（\(-a\)称为a的 “负元”）；
- **加法交换律**：对任意\(a, b \in R\)，\(a + b = b + a\)（环的加法必须交换，这是区别于 “非交换群” 的关键）。

环的乘法要求远弱于加法，仅需满足 “封闭性” 和 “结合律”，这是环与 “域” 的核心差异：

- **乘法封闭性**：对任意\(a, b \in R\)，\(ab \in R\)；
- **乘法结合律**：对任意\(a, b, c \in R\)，\((ab)c = a(bc)\)。

这是环的 “灵魂公理”—— 两种运算并非独立，乘法需对加法满足左右分配律：

- **左分配律**：对任意\(a, b, c \in R\)，\(a(b + c) = ab + ac\)；
- **右分配律**：对任意\(a, b, c \in R\)，\((b + c)a = ba + ca\)。

（若环不满足交换律，左、右分配律需分别验证；若满足交换律，两者等价。）

### 整除

**定义**：$设 f(x), g(x) \in F[x]，若存在一个多项式 h(x) \in F[x]，使得f(x) = g(x) \cdot h(x)
则称 g(x) **整除** \(f(x)\)，记为 \(g(x) \mid f(x)\)。$

**相伴**:=互相整除。（相差c倍，c$\in$ K域）

**\(f(x) $\sim$ g(x) $\iff$ f(x)$ \mid$ g(x)\) 且 \(g(x)$ \mid$ f(x)\)**

**性质**

- 自反性
- 传递性
- Deg(g)<=Deg(f)
- $若 f(x) \mid g(x) 且 f(x) \mid h(x)，则对任意 u(x), v(x) \in F[x]，有 f(x) \mid [u(x)g(x) + v(x)h(x)]$

**带余除法定理**

设 \(f(x), g(x)  $\in  $P[x]\)，且 \(g(x)  $\neq $ 0\)，则存在唯一的一对多项式 \(q(x), r(x) \in P[x]\)，使得：\( $\boxed{f(x) = g(x)q(x) + r(x)} $\) 其中，\(r(x) = 0\) 或 \( $\deg r(x) < \deg g(x) $\)。

这里，\(q(x)\) 称为 \(f(x)\) 除以 \(g(x)\) 的**商式**，\(r(x)\) 称为**余式**。

**整除的等价判定**

由带余除法可知：\($\boxed{g(x) \mid f(x) \iff 余式 r(x) = 0}$\)

这是判定多项式整除的核心准则 —— 只要用 \(g(x)\) 除 \(f(x)\) 所得余式为零，就说明 \(g(x)\) 整除 \(f(x)\)，反之亦然。

### 因式

**公因式**

设\(f(x), g(x)$ \in)$ P[x]\)，若存在多项式\(d(x) )$\in )$P[x]\)，使得：\(d(x))$ \mid)$ f(x)\)（即\(d(x)\)整除\(f(x)\)）**且** \(d(x))$ \mid )$g(x)\)（即\(d(x)\)整除\(g(x)\)），
则称\(d(x)\)是\(f(x)\)与\(g(x)\)的**公因式**。

**最大公因式**

设\(f(x), g(x) $\in$ P[x]\)，若存在公因式\(d(x) $\in $P[x]\)，满足：
对\(f(x)\)与\(g(x)\)的**任意公因式\(h(x)\)**，都有\(h(x) $\mid$ d(x)\)，
则称\(d(x)\)是\(f(x)\)与\(g(x)\)的**最大公因式**，记为\(d(x) = (f(x), g(x))\)。

**引理(辗转相除法理论依据)**

设 \(f(x) = q(x)g(x) + r(x)\)（带余除法的结果），则：\($\gcd(f(x), g(x)) = \gcd(g(x), r(x))$\)

**辗转相除法**

1. 用次数较高的多项式除以较低的（若\($\deg(f(x)) < \deg(g(x))$\)，则交换两者），得余式$\(r_1(x)\)：\(f(x) = q_1(x)g(x) + r_1(x)\)（\(r_1(x) = 0\)或\(\deg(r_1(x)) < \deg(g(x))\)）$；
2. 若\(r_1(x) $\neq$ 0\)，用\(g(x)\)除以\(r_1(x)\)，得余式\(r_2(x)\)：$\(g(x) = q_2(x)r_1(x) + r_2(x)\)$$（\(r_2(x) = 0\)或\(\deg(r_2(x)) < \deg(r_1(x))\)）$；
3. 重复上述过程，余式次数严格递减，最终必存在$\(r_k(x) \neq 0\)，使得\(r_{k-1}(x) = q_k(x)r_k(x) + 0\)$；
4. 此时\(r_k(x)\)就是\(f(x)\)与\(g(x)\)的一个最大公因式，将其首项系数化为 1，即得首一最大公因式。

**(f(x), g(x))** 通常用来表示多项式 \(f(x)\) 和 \(g(x)\) 的**首项系数为 1 的最大公因式**。

**性质**

- 任意一对多项式f(x),g(x)都有最大公因式d(x),且存在u(x),v(x),使得，u(x)f(x)+v(x)g(x)=d(x)(**贝祖定理**)
- 首一最大公因式不随数域扩大而改变。
- 整除性不随数域扩大而改变。

**互素**

设 \(f(x), g(x) $\in$ P[x]\)（\(P[x]\) 表示数域 P 上的一元多项式环），若 \(f(x)\) 和 \(g(x)\) 的**首一最大公因式为 1**，即\($\boxed{(f(x), g(x)) = 1}$\)
则称 \(f(x)\) 与 \(g(x)\)**互素**（或 “互质”）。

**定理**：
\(f(x)\) 与 \(g(x)\) 互素 <=> 存在 \(u(x), v(x) $\in$ P[x]\)，使得\($\boxed{1 = u(x)f(x) + v(x)g(x)}$\)

**互素性**不随数域扩大而改变。

若 \((f(x), g(x)) = 1\)，且 \(f(x) $\mid$ g(x)h(x)\)，则 \(f(x) $\mid $h(x)\)

若$ f_1(x) \mid g(x)，f_2(x) \mid g(x)，且 (f_1(x), f_2(x)) = 1，则 f_1(x)f_2(x) \mid g(x)$

若 \((f(x), g(x)) = 1\)，\((f(x), h(x)) = 1\)，则 \((f(x), g(x)h(x)) = 1\)

### 不可约多项式（类比素数）

**定义**

f(x)$\in$ K[X],deg(f)>0,如果f(x)的因式只有零次多项式和f(x)的相伴元，则称之为不可约多项式。否则，可约。

**等价**

- 数域上的任意多项式与任意一个不可约多项式要么互素要么整除。
- 任意一个不可约多项式整除两个多项式乘积，那么至少整除其中一个。
- 任意一个不可约多项式不能分解为两个次数比它低的两个多项式的成绩
- 不可约多项式
- 👆上面四个等价

**唯一因式分解定理**：K[X]上的次数大于0的多项式都可以唯一分解为K上有限多个不可约多项式的乘积。（类比算术基本定理）

**标准分解式**

对任意\(f(x)$ \in$  F[x]\)（$ \(\deg f(x) \geq 1$ \)），可将其分解式 “标准化”：

1. 取每个不可约因式的 “首一多项式”（首项系数为 1 的相伴多项式），例如将\(2x + 2\)化为\(x + 1\)；
2. 将相同的不可约因式合并为幂次形式。

最终得到**标准分解式**：$ \(f(x) = a \cdot p_1^{e_1}(x) \cdot p_2^{e_2}(x) \cdot \dots \cdot p_k^{e_k}(x)\)$ 

若存在正整数 k，使得：

- \(p(x)^k $\mid $ f(x)\)（读作 “\(p(x)\) 的 k 次方整除 \(f(x)\)”，即 \(f(x)\) 能被 \(p(x)^k\) 整除，商仍是 P 上的多项式）；
- 且  $\(p(x)^{k+1} \nmid f(x)\) $（“不整除”，即 \(f(x)\) 不能被 \(p(x)\) 的 \(k+1\) 次方整除）；

则称 \(p(x)\) 是 \(f(x)\) 的 **k 重因式 **。特别地，

k=0,非因式

k=1，单因式

k>=2,重因式

### 多项式数域中的根

**定义**

**根：**设f(x)是K[X]，如果c $\in$ K,使得f(c)=0,则称c为f(x)在K上的一个根。

**性质**

在K[X]中，**x-c|f(x)<=>c是f(x)在K上的一个根**。(Bezout定理)（k重因式，k重根）

K[X]中的n（n>=0）次多项式在K中至多有n个根。

deg(f,g)<=n,如果K中存在n+1个不同的数c_i使得f=g(c_i),那么f=g。

### 代数基本定理

**背景**

设f(x)$\in$C(x),deg(f)=n>0,复数域内有根吗？

**引理**

（**Liuovill定理**）在复平面上解析且模有界的函数一定是常值函数。

**内容**

对于任意一个 **次数≥1 的复系数多项式**
\($P(z) = a_n z^n + a_{n-1} z^{n-1} + \dots + a_1 z + a_0$\)
（其中 \(n $\geq$ 1\)，\($a_n \neq 0$\)，系数 \($a_n, a_{n-1}, \dots, a_0$\) 均为复数），
**至少存在一个复数根**（即存在某个复数 \($z_0$\)，使得 \(P($z_0$) = 0\)）。

**推论**

复数域中次数大于1的多项式有一次因式，从而一定**可约**。

复数域中，不可约多项式**只有一次多项式**。

### 实数域上的不可约多项式

**性质**

- 复根成对定理 若 \($\alpha = a + bi$\)（\(b $\neq$ 0\)）是实系数多项式的根，则其 “共轭复数” \($\overline{\alpha} = a - bi$\) 也一定是该多项式的根。
- 只有一次多项式和判别式小于零的二次多项式。

### 有理数域上的不可约多项式

**定义**

**本原多项式： **系数的最大公因子（gcd）为 \($\pm1$\) 的非零整系数多项式。

**性质**

- Q[X]中次数大于零的多项式f(x)存在一个本原多项式与它相伴。

- 在Q[X]中两个本原多项式，它们相伴$\leftrightarrow$它们要么相等要么相反。

- （高斯引理）两个本原多项式的乘积是本原多项式。

- 次数大于零的本原多项式g在Q上可约$\leftrightarrow$ g可以分解为两个次数比g低的本原多项式的乘积。

- 对于**本原多项式**(整系数) \(g(x) = $b_nx^n + \dots + b_1x + b_0\)（\(b_n, \dots, b_0 \in \mathbb{Z}$\)，gcd\(($b_n, \dots, b_0$) = 1\)），若 \($\frac{p}{q}$\)（\(p, q $\in$$ \mathbb{Z}$\)，且 gcd\((p, q) = 1\)，即 “既约分数”）是 \(g(x)\) 的一个有理根，则必有：

  1. 分子 p 是常数项 \(b_0\) 的约数（\($p \mid b_0$\)）；
  2. 分母 q 是首项系数 \(b_n\) 的约数（\($q \mid b_n$\)）。

- 二次或者三次的整系数多项式f，没有有理根$\leftrightarrow$ f在Q上不可约。

- （Eisenstein判别）f是一个次数大于零的整系数多项式，如果存在一个素数满足：

  1. p不能整除首项，其余可以
  2. $p^2$ 不能整除常数项

  那么f在Q上不可约。

- Q[X]中存在任意次数的不可约多项式。

- 次数大于零的整系数多项式f在Q上不可约$\leftrightarrow$ f(x+-1)Q上不可约。

#### 数域的定义

数域是一类具有 “封闭性” 的数集，需满足以下**4 个条件**（设数集为F，且F中至少包含两个不同的数）：

1. **对加法封闭**：若\(a, b$ \in$ F\)，则\(a + $b \in $F\)；
2. **对减法封闭**：若\(a, b $\in $F\)，则\(a - b $\in $F\)；
3. **对乘法封闭**：若\(a, b $\in $F\)，则\(a $\times $b $\in $F\)；
4. **对除法封闭（除数非零）**：若\(a, b $\in$ F\)且\(b$ \neq $0\)，则\($\frac{a}{b} $$\in $F\)。

有理数域是最小的数域。

### 向量与线性空间

#### 线性空间（**线性空间 = 带有数乘结构的交换群（且数乘来自域）** **线性空间 = 域上的模**）

**定义**

第一类：向量加法的公理（集合内定义 “加法” 运算，记为\($\vec{\alpha} $+ $\vec{\beta}$\)）

对任意$\vec{\alpha}, \vec{\beta}, \vec{\gamma} \in V（V为向量集合），加法需满足：$

- **交换律**：$\vec{\alpha} + \vec{\beta} = \vec{\beta} + \vec{\alpha}（加法顺序不影响结果）；$
- **结合律**：$(\vec{\alpha} + \vec{\beta}) + \vec{\gamma} = \vec{\alpha} + (\vec{\beta} + \vec{\gamma})（多个向量相加，分组顺序不影响结果）$；
- **零向量存在性**：$存在唯一的\(\vec{0} \in V\)，使得对任意\(\vec{\alpha} \in V\)，有\(\vec{\alpha} + \vec{0} = \vec{\alpha}\)（\(\vec{0}\)称为 “零向量”，类似数的 “0”）；$
- **负向量存在性**：$对任意\(\vec{\alpha} \in V\)，存在唯一的\(\vec{\beta} \in V\)，使得\(\vec{\alpha} + \vec{\beta} = \vec{0}\)（\(\vec{\beta}\)称为\(\vec{\alpha}\)的 “负向量”，记为\(-\vec{\alpha}\)，类似数的 “相反数”）。$

第二类：数量乘法的公理（数域与集合间定义 “数乘” 运算，记为\(k\vec{\alpha}\)）

$对任意\(\vec{\alpha}, \vec{\beta} \in V\)和任意\(k, l \in \mathbb{F}\)（\(\mathbb{F}\)为数域），数乘需满足：$

- **数乘结合律**：$\(k(l\vec{\alpha}) = (kl)\vec{\alpha}\)（数量的乘法与数乘可交换顺序）；$
- **分配律 1（对数的加法）**：$\((k + l)\vec{\alpha} = k\vec{\alpha} + l\vec{\alpha}\)（数量相加后数乘，等于分别数乘后相加）；$
- **分配律 2（对向量的加法）**：$\(k(\vec{\alpha} + \vec{\beta}) = k\vec{\alpha} + k\vec{\beta}\)（向量相加后数乘，等于分别数乘后相加）；$
- **单位元律**：$\(1 \cdot \vec{\alpha} = \vec{\alpha}\)（数域中的 “1” 与向量数乘，结果仍为原向量，类似数的 “1×a=a”）。$

**线性空间性质**

**线性空间同构**：$设 V 和 W 是数域 \mathbb{F}（如实数域 \mathbb{R}、复数域 \mathbb{C}）上的两个线性空间。若存在一个映射 \sigma: V \to W，满足以下三个条件，则称 V 与 W 是同构的，记为 V \cong W，且 \(\sigma\) 称为 V 到 W 的**同构映射**：$

- 保运算
- 单射
- 满射（一一对应）

**同构的性质**：

1. **保持零元素**：\(\sigma(0_V) = 0_W\)（\(0_V\) 是 V 的零元素，\(0_W\) 是 W 的零元素）。
2. **保持负元素**：对任意 \(\alpha \in V\)，\(\sigma(-\alpha) = -\sigma(\alpha)\)。
3. **保持线性组合**：对任意 \(\alpha_1, \dots, \alpha_n \in V\) 和 \(k_1, \dots, k_n \in \mathbb{F}\)，有：\(\sigma(k_1\alpha_1 + \dots + k_n\alpha_n) = k_1\sigma(\alpha_1) + \dots + k_n\sigma(\alpha_n)\)。
4. **保持线性相关性 / 无关性**：V 中向量组 \(\alpha_1, \dots, \alpha_n\) 线性无关（或相关），当且仅当 W 中对应向量组 \(\sigma(\alpha_1), \dots, \sigma(\alpha_n)\) 线性无关（或相关）。
5. **保持基与维数**：若 \(\alpha_1, \dots, \alpha_n\) 是 V 的一组基，则 \(\sigma(\alpha_1), \dots, \sigma(\alpha_n)\) 是 W 的一组基，因此 \(\dim V = \dim W\)（维数相等是同构的**必要条件**）。

**核心定理（同构的判定）**

对**数域 \($\mathbb{F}$\) 上的有限维线性空间** V 和 W，以下命题等价：

1. \(V $\cong$ W\)（同构）；
2. \(\dim V = \dim W\)（维数相等）；
3. 存在从 V 到 W 的双射线性映射（即同构映射）。

**推论**：数域 \(\mathbb{F}\) 上，所有 n 维线性空间都与 \(\mathbb{F}^n\)（\(\mathbb{F}\) 上的 n 元有序数组空间）同构。

**维数相同的线性空间在代数意义上完全等价，可通过同构映射将一个空间的问题转化为另一个空间（如 \($\mathbb{F}^n$\)）的问题来解决**

**线性空间的同构是一个等价关系，此时等价类就是同构类，所有同构类组成的集合就是一个划分。通过同构给数域K上的线性空间一个划分。**

**陪集，商集，商空间**：

陪集是 “给线性空间中的元素按‘子空间’分类” 的工具，核心是 “固定一个子空间，用‘元素 + 子空间’的方式生成一类元素”。

定义（针对线性空间）：

设 `V` 是数域 `F` 上的线性空间，`W` 是 `V` 的**子空间**（比如 `V=R³`，`W` 是 `xy` 平面，即 `W={(x,y,0)|x,y∈R}`），任取 `v₀ ∈ V`（比如 `v₀=(0,0,1)`），则：

- **加法陪集（左陪集，线性空间中常用）**：`v₀ + W = { v₀ + w | w ∈ W }`
  （因为线性空间加法可交换，左陪集 = 右陪集，统称 “陪集”）

- 陪集是 “平移后的子空间”：子空间 `W` 本身也是一个陪集（取 `v₀=0`，则 `0 + W = W`）；其他陪集都是 `W` 沿 `v₀` 平移得到的，与 `W` 平行且不相交。
- 所有陪集构成对 `V` 的 “划分”：`V` 中每个元素恰好属于一个陪集（比如 3D 空间中，每个点恰好属于一个 “高度固定的平行平面”）。

商集是把 “所有陪集” 当作 “新元素”，构成的一个新集合。

定义（针对线性空间）：

设 `V` 是线性空间，`W` 是 `V` 的子空间，所有以 `W` 为 “模板” 的陪集（即 `{ v + W | v ∈ V }`）构成的集合，称为 `V` 关于 `W` 的商集，记为 `V/W`（读作 “V 模 W” 或 “V 除以 W”）。

**商集只是 “一堆陪集”，而商空间是给商集赋予 “加法” 和 “数乘” 运算，让它也成为一个线性空间 —— 这才是线性代数中真正有用的概**念。

**定义：**

设 `V` 是数域 `F` 上的线性空间，`W` 是 `V` 的子空间，对商集 `V/W` 定义以下**线性运算**：

1. 陪集加法

   ：对任意

    

   ```

   ```

   ，定义

    

   ```
   (v₁ + W) + (v₂ + W) = (v₁ + v₂) + W
   ```

   

   （类比：“高度 1 的平面” + “高度 2 的平面” = “高度 3 的平面”）

2. 数乘

   ：对任意

    

   ```
   k ∈ F
   ```

   ，

   ```
   v + W ∈ V/W
   ```

   ，定义

    

   ```
   k·(v + W) = (k·v) + W
   ```

   

   （类比：“2 倍高度 1 的平面” = “高度 2 的平面”）

此时，`V/W` 关于这两种运算构成数域 `F` 上的线性空间，称为 `V` 关于 `W` 的商空间。

**核心意义：“降维”—— 去掉 “冗余维度”**

#### 线性映射（同态映射）

##### 定义

**线性空间**：

V 是数域 F 上的**线性空间**，W 也是数域 F 上的**线性空间**，如果有一个**对应法则**

σ:V→W

把 V 中每个向量 α 对应到 W 中唯一向量 σ(α)，就称 σ 是从 V 到 W 的一个**映射**。

如果这个映射 σ 满足下面两条，就叫**线性映射**：

1. **加法保持**

   σ(α+β)=σ(α)+σ(β),∀α,β∈V

   

2. **数乘保持**

   σ(kα)=kσ(α),∀α∈V, ∀k∈F

 V上到自身的线性映射叫**线性变换**。

**Hom(V,W)**:

设 V 和 W 都是数域 F 上的线性空间。

记 L(V,W) 为所有从 V 到 W 的线性映射 σ:V→W 构成的集合。

**线性映射加法**：

设 σ,τ∈L(V,W)

(σ+τ)(α)=σ(α)+τ(α),∀α∈V

**数乘**：

设 σ,τ∈L(V,W)

(σ+τ)(α)=σ(α)+τ(α),∀α∈V

**乘法（复合）**：

σ:U→V,τ:V→W

复合映射 τσ=τ∘σ:U→W，定义：

(τσ)(α)=τ(σ(α))

**投影变换**：

设 V 是线性空间，P:V→V 是线性变换，如果满足

P2=P

即对任意 α∈V

P(P(α))=P(α)

就称 P 是**投影变换**（幂等线性变换）。

**核 \**kerσ\** 是 \**V\** 的子空间**

kerσ={α∈V∣σ(α)=0}

**像 \**Imσ\** 是 \**W\** 的子空间**

Imσ=σ(V)={σ(α)∣α∈V}

σ(0)=0

**把负向量映成负像**

σ(−α)=−σ(α)

**线性组合保持**

σ(k1α1+⋯+ksαs)=k1σ(α1)+⋯+ksσ(αs)

**线性相关组的像仍线性相关**

若 α1,…,αs 线性相关，则

σ(α1),…,σ(αs) 也线性相关

若 σ 是**单射**，且 α1,…,αs 线性无关，则

σ(α1),…,σ(αs) 线性无关

设向量 α∈V：

1. α 在旧基 ε 下坐标：X
2. α 在新基 η 下坐标：Y

则**坐标变换公式**：

X=AY,Y=A−1X

**不同基之间的矩阵：**

线性变换

（同空间、一组基变换）：换基相似A′=$P^{-1}$AP

线性映射

（两空间、两组基变换）：换基相抵A′=$Q{-1}$AP

#### 矩阵的迹

##### 定义

**迹：**

矩阵 A 的**迹**记作 tr(A)，定义为：

tr(A)=a11+a22+⋯+ann

直白说：**主对角线元素之和**。

##### 性质

**线性性：**

tr(A+B)=tr(A)+tr(B)

tr(kA)=ktr(A), k∈P

**转置迹不变：**

tr(AT)=tr(A)

**乘积交换迹：**

若 Am×n,Bn×m，则：

tr(AB)=tr(BA)

#### 特征值

##### 定义

**特征值与特征向量：**

设 F 为**域**，V 是 F 上**有限维线性空间**，

A:V→V 是 V 上的一个**线性变换**。

 特征值

若存在 λ∈F，使得存在**非零向量**ξ∈V，满足

Aξ=λξ

则称 λ 为线性变换 A 的一个**特征值**。

特征向量

满足上式的**非零向量** ξ 称为 A 属于（对应于）特征值 λ 的**特征向量**。

**特征子空间：**

设 F 为域，V 是 F 上有限维线性空间，

A∈EndF(V) 是线性变换。

对 λ∈F，定义：

Vλ={ξ∈V∣Aξ=λξ}

称 Vλ 为 A 对应于特征值 λ 的**特征子空间**

**特征矩阵与特征多项式：**

设 F 为域，V 是 n 维 F- 线性空间，A∈End(V)，

取定一组基，A 的矩阵为 A∈Mn(F)，E 为 n 阶单位矩阵。

λE−A

称为方阵 A 的 **特征矩阵**。

- 元素属于**一元多项式环** F[λ]；
- 是 n 阶 F[λ] 上的方阵；
- 区别：A−λE 也常用，仅差符号。
- 特征矩阵是**含参矩阵**，不是数域矩阵，元素是多项式。

f(λ)=det(λE−A)

称为矩阵 A 的**特征多项式**。

- f(λ)∈F[λ]，是**首一 \**n\** 次多项式**；
- 有限维下：**线性变换 \**A\** 的特征多项式** 就定义为其任意基下矩阵的特征多项式；
- 良定性：相似矩阵特征多项式相等，故与基选取无关，是线性变换的**内蕴性质**。

**代数重数：**

特征多项式因式分解：

f(λ)=∣λE−A∣=(λ−λ1)m1(λ−λ2)m2⋯(λ−λs)ms

对特征值 λ0：

λ−λ0)因式的指数 \**m\**，就叫 \**λ0\** 的代数重数**。

**几何重数：**

对应齐次方程组：

(λ0E−A)x=0

它的**解空间（特征子空间）的维数**，称为 λ0 的几何重数。

##### 性质

**不同特征值的特征子空间直和：**

若 λ1,…,λk 两两不同，则

Vλ1+⋯+Vλk=Vλ1⊕⋯⊕Vλk

即属于不同特征值的特征向量线性无关。

**有限维，矩阵与线性变换的特征值相等，特征向量与坐标一一对应 ，特征值子空间同构**

**特征多项式展开：**

设 A∈Mn(F)，**特征多项式**

f(λ)=∣λE−A∣=det(λE−A)

这是**首一 \**n\** 次多项式**，统一展开形式：

f(λ)=λn−tr(A)λn−1+σ2(A)λn−2−⋯+(−1)n−1σn−1(A)λ+(−1)n∣A∣

设

A=(aij)n×n

记 σk(A) 为：A的所有 k阶主子式之和。

λn−k 系数：(−1)kσk(A)

#### 对角化

##### 定义  

**线性变换可对角化：**

F 为域，V 是 F 上 n 维线性空间， 

A∈End(V)。

若存在 V 的一组基 ε1,ε2,…,εn，

使得线性变换 A 在这组基下的**矩阵是对角矩阵**，

则称：A 是**可对角化**的线性变换。

**矩阵可对角化：**

设 A 是数域 F 上 n 阶方阵：

若**存在**数域 F 上的**可逆矩阵 \**P\****，使得：

$P^{−1}AP=Λ$

其中 Λ 为**对角矩阵**，

则称：***\*A\** 在数域 \**F\** 上可对角化**。

##### 性质

**线性变换可对角化<=>存在一组特征向量基 <=>全空间分解为特征子空间的直和<=>空间维度等于各个特征子空间的维度和<=>代数重数等于几何重数**

**矩阵可对角化<=>矩阵有n个线性无关的特征向量，且特征向量的矩阵就是可逆矩阵P**

#### 不变子空间

##### 定义

**不变子空间：**

设 V 是数域 F上的线性空间，A:V→V 是线性变换，W 是 V 的子空间。

若 ***\*W\** 中任意向量被 \**A\** 作用后，结果仍在 \**W\** 中**，即

∀α∈W, Aα∈W

则称 W 是 A 的**不变子空间**（也叫 A- 子空间）。把大空间拆成小空间，让线性变换 “分块对角化”

**分块对角矩阵：**

形式：**只有主对角线上有方块，其余全是零块**
$$
\begin{pmatrix}
A&0&0\\
0&B&0\\
0&0&C
\end{pmatrix}
$$




- A,B,C 是小方阵
- 非对角位置全部为零矩阵

##### 性质

**零空间和V是所有线性变换的不变子空间**

**特征子空间，核空间，像空间都是不变子空间**

**如果A,B $\in$ Hom(V,V)，AB=BA，那么B的 特征子空间，核空间，像空间都是A的不变子空间**

**不变子空间的和与交还是不变子空间**

**向量组生成的空间W，如果向量组里的每个向量线性变换后还在W，则W是不变子空间**

**不变子空间的直和分解 <==>矩阵分块对角**

## 拓展

### 群环域

| 结构 | 运算数量     | 核心运算性质                                                 | 关键差异点                    | 典型例子                 |
| ---- | ------------ | ------------------------------------------------------------ | ----------------------------- | ------------------------ |
| 群   | 1 个（・）   | 运算构成阿贝尔群（或非阿贝尔群）                             | 只有 1 个运算，无 “乘法” 概念 | (ℤ, +), (ℝ*, ×)          |
| 环   | 2 个（+, ×） | 加法构成阿贝尔群；乘法满足结合律、分配律；无乘法逆元要求     | 有乘法但无逆元，可能无单位元  | (ℤ, +, ×), (2ℤ, +, ×)    |
| 域   | 2 个（+, ×） | 加法构成阿贝尔群；非零元素乘法构成阿 贝尔群（有单位元、逆元）；乘法交换 | 乘法除 0 外可逆，有单位元 1   | (ℚ, +, ×), (ℝ, +, ×), ℤ₅ |

#### 群

**定义**

设 G 是非空集合，“・” 是 G 上的二元运算，若满足以下 4 条公理，则 (G,・) 称为**群**：**对称**

| 公理名称      | 具体要求                                                     | 示例（整数加法群 (ℤ, +)）                 |
| ------------- | ------------------------------------------------------------ | ----------------------------------------- |
| 1. 封闭性     | 对任意 a, b ∈ G，都有 a・b ∈ G                               | 任意两个整数相加，结果还是整数（3+5=8∈ℤ） |
| 2. 结合律     | 对任意 a, b, c ∈ G，都有 (a・b)・c = a・(b・c)               | (2+3)+4 = 2+(3+4)                         |
| 3. 存在单位元 | 存在 e ∈ G，使得对任意 a ∈ G，都有 e・a = a・e = a           | 单位元是 0（0+a = a+0 = a）               |
| 4. 存在逆元   | 对任意 a ∈ G，存在 b ∈ G，使得 a・b = b・a = e（b 称为 a 的逆元，记为 a⁻¹） | 整数 a 的逆元是 - a（a + (-a) = 0）       |

**常见群的例子**

- 整数加法群 (ℤ, +)、有理数加法群 (ℚ, +)、实数加法群 (ℝ, +)；
- 非零实数乘法群 (ℝ*, ×)（单位元是 1，a 的逆元是 1/a）；
- 模 n 剩余类加法群 (ℤₙ, +)（比如ℤ₃={[0],[1],[2]}，[1]+[2]=[0]）。

**若群的运算还满足**交换律**（对任意 a,b∈G，a・b = b・a），则称为**阿贝尔群**（或交换群）**

#### 环

**定义**

设 R 是非空集合，“+” 和 “×” 是 R 上的两个二元运算，若满足以下 ，则 (R, +, ×) 称为**环**：

| 公理类别             | 具体要求                                                     | 示例（整数环 (ℤ, +, ×)）                     |
| -------------------- | ------------------------------------------------------------ | -------------------------------------------- |
| 加法公理（阿贝尔群） | 1. 加法封闭性；2. 加法结合律；3. 加法单位元（记为 0）；4. 加法逆元；5. 加法交换律 | 同整数加法群，0 是加法单位元，-a 是 a 的逆元 |
| 乘法公理             | 6. 乘法封闭性；7. 乘法结合律                                 | 整数相乘仍为整数，(2×3)×4=2×(3×4)            |
| 分配律               | 8. 左分配律：a×(b+c) = a×b + a×c；9. 右分配律：(b+c)×a = b×a + c×a | 2×(3+4)=2×3+2×4；(3+4)×2=3×2+4×2             |

**环的关键特点**

- 乘法不需要 “单位元”：比如 “所有偶数构成的环 (2ℤ, +, ×)”，没有乘法单位元（不存在偶数 e 使得 e×a = a×e = a）；
- 乘法不需要 “逆元”：比如整数环ℤ中，除了 1 和 - 1，其他元素（如 2）没有乘法逆元（2×x=1 的解 x=1/2∉ℤ）；
- 乘法可能不满足交换律：若乘法满足交换律（a×b = b×a），则称为**交换环**（如ℤ, ℚ, ℝ都是交换环）；不满足的称为非交换环（如 “n 阶矩阵环”，矩阵乘法不交换）。

**常见环的例子**

- 整数环 (ℤ, +, ×)、有理数环 (ℚ, +, ×)、实数环 (ℝ, +, ×)；
- 多项式环（如ℤ[x]：所有整系数多项式构成的环，加法是多项式相加，乘法是多项式相乘）；
- 模 n 剩余类环 (ℤₙ, +, ×)（如ℤ₄={[0],[1],[2],[3]}，[2]×[2]=[0]）。

#### 域

**定义**

设 F 是非空集合，“+” 和 “×” 是 F 上的两个二元运算，若满足以下 9 条公理，则 (F, +, ×) 称为**域**：

| 公理类别                         | 具体要求                                                     | 示例（有理数域 (ℚ, +, ×)）                          |
| -------------------------------- | ------------------------------------------------------------ | --------------------------------------------------- |
| 加法公理（阿贝尔群）             | 同环的加法公理（封闭、结合、单位元 0、逆元、交换）           | 有理数相加仍为有理数，0 是单位元，-a 是逆元         |
| 乘法公理（非零元素构成阿贝尔群） | 1. 乘法封闭性；2. 乘法结合律；3. 乘法单位元（记为 1，1≠0）；4. 乘法逆元（对任意 a≠0，存在 a⁻¹∈F 使 a×a⁻¹=1）；5. 乘法交换律 | 非零有理数相乘仍为非零有理数，1 是单位元，a⁻¹=1/a∈ℚ |
| 分配律                           | 同环的分配律（左分配、右分配）                               | a×(b+c)=a×b+a×c                                     |

**域的关键特点**

- 必须有**乘法单位元 1**（且 1≠0）；

- 除了 0 之外，**每个元素都有乘法逆元**（这是域与环的核心区别）；

- 乘法必须**交换**（非交换的 “域” 称为 “除环” 或 “体”，如四元数体，但不是域）。

- 对任意域 F，考虑乘法单位元 1 的倍数序列：\(1 $\cdot $ 1 = 1\)，\(2  $\cdot  $1 = 1+1\)，\(3  $\cdot $ 1 = 1+1+1\)，... 有且只有两种可能：

  - 所有 \(n  $\cdot $ 1\)（\(n $ \in \mathbb{Z}^+ $\)）都不等于加法单位元 0，此时称域 F 的**特征为 0**（记为 \( $\text{char}(F) = 0 $\)），例如有理数域 \(\mathbb{Q}\)、实数域 \(\mathbb{R}\)、复数域 \(\mathbb{C}\)；
  - 存在最小的正整数 p，使得 \(p  $\cdot  $1 = 0\)，此时称域 F 的**特征为 p**（记为 \( $\text{char}(F) = p $\)），且这个最小的 p 一定是**素数**（关键！）。

  对任意域 F，考虑：

  \{$n\cdot1 \mid n\in\mathbb{Z}$\}

  这个集合：

  - 在 char 0 时 → 同构于 ℤ，生成 ℚ
  - 在 char p 时 → 同构于$ \mathbb{Z}/p\mathbb{Z}$

- > **域的特征，刻画了“1 在加法下的周期性”。**

  > **特征为 0 ⇒ 无限直线；特征为 p ⇒ 模 p 的循环世界。**

  > **“p 必须是素数”反映了域“无零因子”的本质。**

  > **任何域，都是从 ℚ 或**$ \mathbb{F}_p $**生长出来的。**

- 数域的特征为0 。

**常见域的例子**

- 有理数域ℚ、实数域ℝ、复数域ℂ（最经典的数域）；
- 模 p 剩余类域ℤₚ（p 是质数，比如ℤ₅={[0],[1],[2],[3],[4]}，[2] 的逆元是 [3]，因为 [2]×[3]=[6]=[1]）；
- 注意：ℤ₄不是域（[2]≠0，但 [2] 没有乘法逆元，因为 [2]×[x]=[1] 在ℤ₄中无解）。



### 模的剩余类

#### 定义

对于固定的模 m，所有与整数 r 关于模 m 同余的整数，构成一个 “模 m 的剩余类”，记为 \($[r]_m$\)（或简记为 \([r]\)，当模 m 明确时）。

所有模 m 的剩余类构成的集合记为\($\mathbb{Z}/m\mathbb{Z}$\)（或\($\mathbb{Z}_m$\)

对任意两个剩余类\($\overline{a}, \overline{b} \in \mathbb{Z}/m\mathbb{Z}$\)，定义：

- **加法**：\($\overline{a} + \overline{b} = \overline{a + b}$\)（即 “类的和 = 和的类”）；
- **乘法**：\($\overline{a} \times \overline{b} = \overline{a \times b}$\)（即 “类的积 = 积的类”）。

#### 性质

- 当m是素数（记为p）时：\($\mathbb{Z}/p\mathbb{Z}$\)是 “域”.m是合数，一定不是域。
