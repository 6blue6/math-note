## 前置知识

### 常用不等式

**三角不等式（Triangle Inequality）**

**形式**：对任意实数 / 复数 / 向量 \(a, b\)，有 \(|a + b| $\leq$ |a| + |b|\) 等号成立当且仅当 a 与 b 同方向（实数：\(ab $\geq$ 0\)；复数：辐角相同；向量：共线且同向）。

**均值不等式 (Mean Inequality)**

调和均值（HM）\($\leq$\) 几何均值（GM）\($\leq$\) 算术均值（AM）\($\leq$\) 平方均值（QM），即
$(\frac{n}{\sum \frac{1}{a_i}} \leq \sqrt[n]{\prod a_i} \leq \frac{\sum a_i}{n} \leq \sqrt{\frac{\sum a_i^2}{n}})$

**伯努利不等式（Bernoulli's Inequality）**

基础版：$对实数 x > -1 且整数 n \geq 1，有 (1 + x)^n \geq 1 + nx)$；

加强版（实数指数）：$对实数 x > -1 且 \alpha \leq 0 或 \alpha \geq 1有 ((1 + x)^\alpha \geq 1 + \alpha x；若 0 < \alpha < 1，则不等式反向（(1 + x)^\alpha \leq 1 + \alpha x）。$

- **等号成立条件**：\(x = 0\) 或 \(n = 1\)（基础版）；\(x = 0\) 或 \(\alpha = 0, 1\)（加强版）。
- **用途**：数列极限的估计（如证明 \((1 + $\frac{1}{n})^n$\) 单调递增）、幂函数的放缩。

**柯西 - 施瓦茨不等式（Cauchy-Schwarz Inequality）**

**核心形式（内积空间版）**：对内积空间 V 中的任意向量 \(u, v\)，有
$|(u, v)|^2 \leq (u, u) \cdot (v, v)$

**符号函数（Sign Function，\($\text{sgn}(x)$\)）**

- **定义**：$\\text{sgn}(x) = \begin{cases} 1, & x > 0 \\ 0, & x = 0 \\ -1, & x < 0 \end{cases}$\)；
- **特点**：在 \(x=0\) 处间断（第一类间断点），无法用单个初等函数表示（需注意：\($\text{sgn}(x) = \frac{x}{|x|}\)（\(x \neq 0\)$），但仍需分段补充 \(x=0\) 的定义）；
- **应用**：描述物理量的方向（如速度、力的正负）、控制理论（开关信号）。

**狄利克雷函数（Dirichlet Function，\(D(x)\)）**

- **定义**：$\(\text{Dirichlet}\) 函数是 “处处间断” 的典型例子，定义为：\(D(x) = \begin{cases} 1, & x \in \mathbb{Q} \text{（有理数）} \\ 0, & x \in \mathbb{R}\setminus\mathbb{Q} \text{（无理数）} \end{cases}\)$；

- 性质

  ：

  - 周期性：任何正有理数都是其周期（无最小正周期）；
  - 可积性：在任意区间上**黎曼不可积**（振幅无法通过细分区间减小），但**勒贝格可积**（积分值为 0）；

- **意义**：作为 “反例” 推动积分理论的发展（从黎曼积分到勒贝格积分），揭示函数的复杂性。

**黎曼函数（Riemann Function，\(R(x)\)）**

- **定义**：\($\text{Riemann}$\) 函数是 “仅在无理点连续、有理点间断” 的特殊函数，定义为：$\(R(x) = \begin{cases} \frac{1}{q}, & x = \frac{p}{q} \text{（既约分数，} q > 0\text{）} \\ 0, & x \in \mathbb{R}\setminus\mathbb{Q} \text{或} x=0,1 \end{cases}\)（通常定义在 \([0,1]\) 上，可周期延拓）$；
- **性质**：在 \([0,1]\) 上**黎曼可积**，积分值为 0；
- **意义**：证明 “黎曼可积函数未必连续”，是分析学中研究函数连续性与可积性的重要例子。

## 实数理论

### 实数的构造

- **戴德金分割（Dedekind Cut）**

  由戴德金（Richard Dedekind）于 1872 年提出，核心是 “用有理数的分割定义实数”，避免直接涉及 “极限” 概念，逻辑上最严谨。

  **分割的定义**：将有理数集\($\mathbb{Q}$\)划分为两个非空子集A和B，满足：

  1. 覆盖性：\(A $\cup$ B = $\mathbb{Q}$\)，且\(A$ \cap$ B = $\emptyset$\)；
  2. 有序性：对任意\(a $\in$ A\)、\(b$ \in$ B\)，都有\(a < b\)（A中所有元素小于B中所有元素）。

  **分割的分类**：根据A是否有最大元、B是否有最小元，有理数的分割分为 3 类：

  - 类型 1：A有最大元\(a_0\)（则B无最小元），对应有理数\(a_0\)；
  - 类型 2：B有最小元\(b_0\)（则A无最大元），对应有理数\(b_0\)；
  - 类型 3：$A无最大元且B无最小元（“中间有空隙”），对应**无理数**（如\sqrt{2}的分割：A=\{x \in \mathbb{Q} \mid x \leq 0 \text{ 或 } x^2 < 2\}，B=\{x \in \mathbb{Q} \mid x > 0 \text{ 且 } x^2 > 2\}）$。

  **实数的定义**：所有 “有理数的分割” 构成的集合即为实数集\($\mathbb{R}$\)，其中类型 1/2 对应有理数，类型 3 对应无理数。

- **柯西数列等价类（Cauchy sequences）**

由康托尔（Georg Cantor）于 1872 年提出，核心是 “用有理数的柯西序列定义实数”，更贴合数学分析中 “极限” 的直观理解（因为数列极限是分析学的起点）。

- **柯西序列的定义**：$有理数序列\{x_n\}若满足 “\forall \varepsilon > 0（\varepsilon \in \mathbb{Q}），\exists N \in \mathbb{N}，当(m,n > N时，|x_m - x_n| < \varepsilon”，则称其为$**有理数柯西序列**。
- **有理数的缺陷**：有理数柯西序列的极限不一定是有理数（$如x_n = (1+\frac{1}{n})^n是有理数序列，但极限e不是有理数）$。
- **实数的定义**：将 “等价的有理数柯西序列” 视为同一个实数（等价关系：$\{x_n\} \sim \{y_n\} iff \lim_{n \to \infty} |x_n - y_n| = 0）$，所有等价类构成的集合即为\($\mathbb{R}$\)。

### 性质

**有序域结构**

实数是一个：

- 交换、结合、可逆的**域**
- 并带有一个满足兼容性的**次序**（即 a ≤ b 意味着 b − a 是非负的）

**阿基米德性质（Archimedean Property）**

实数集的 “无界性” 特征，是连接 “有限” 与 “无限” 的关键，表述为：

- $对任意正实数a,b \in \mathbb{R}^+，存在正整数n \in \mathbb{N}，使得na > b。$

### 核心性质（完备性）

| 确界原理（Supremum Principle）           | 非空有上界的实数集必有**最小上界**（上确界）；非空有下界的实数集必有**最大下界**（下确界）。 |
| ---------------------------------------- | ------------------------------------------------------------ |
| 单调有界定理                             | 单调递增且有上界的实数序列必有极限；单调递减且有下界的实数序列必有极限。 |
| 柯西收敛准则                             | 实数序列收敛的充要条件是：该序列是柯西序列（即 “极限存在” 与 “柯西性” 等价）。 |
| 闭区间套定理（Nested Intervals Theorem） | $若\{[a_n, b_n]\}是闭区间序列，满足[a_1, b_1] \supset [a_2, b_2] \supset \cdots且\lim_{n \to \infty} (b_n - a_n) = 0，则存在唯一实数\xi \in \bigcap_{n=1}^\infty [a_n, b_n]。$ |
| 聚点定理（Bolzano-Weierstrass Theorem）  | 有界的实数无限集必有**聚点**（即存在一个点，其任意邻域内都含有该集合的无限多个点）。（有界数列必有子列） |
| 有限覆盖定理（Heine-Borel Theorem）      | 若开区间族\($\mathcal{U}$\)覆盖闭区间\([a,b]\)（即\([a,b] $\subset$$ \bigcup_{U \in \mathcal{U}} U$\)），则存在\($\mathcal{U}$\)的有限子集也覆盖\([a,b]\)。 |

**关键逻辑链**：通常以 “确界原理” 作为 “完备性公理”（原始假设），再依次证明其他 5 个定理与确界原理等价 —— 这是数学分析中 “实数理论” 部分的核心证明训练（如用确界原理证单调有界定理、用闭区间套定理证有限覆盖定理等）。

**Stolz定理**：数列极限的 “洛必达法则”

1. $\frac{\infty}{\infty}型 Stolz 定理（最常用形式）$

设数列\(\{a_n\}\)和\(\{b_n\}\)满足以下 3 个条件：

1. **分母数列严格递增且趋于无穷**：$\(\{b_n\}\)是严格递增数列（即对所有n，\(b_{n+1} > b_n\)），且\(\lim_{n \to \infty} b_n = +\infty\)；$
2. **极限存在或为无穷**：$\(\lim_{n \to \infty} \frac{a_{n+1} - a_n}{b_{n+1} - b_n} = L\)（其中L可以是有限数、\(+\infty\)或\(-\infty\)）；$

则**分子分母数列的极限等于上述差商的极限**： $\lim_{n \to \infty} \frac{a_n}{b_n} = L$

2. \(\frac{0}{0}\)型 Stolz 定理

设数列\(\{a_n\}\)和\(\{b_n\}\)满足以下 3 个条件：

1. **分子分母均趋于 0**：\(\lim_{n \to \infty} a_n = 0\)且\(\lim_{n \to \infty} b_n = 0\)；
2. **分母数列严格递减**：\(\{b_n\}\)是严格递减数列（即对所有n，\(b_{n+1} < b_n\)）；
3. **极限存在或为无穷**：$\(\lim_{n \to \infty} \frac{a_{n+1} - a_n}{b_{n+1} - b_n} = L\)（L为有限数、\(+\infty\)或\(-\infty\)）；$

则同样有：$ \(\lim_{n \to \infty} \frac{a_n}{b_n} = L\)$

### 定义

设函数 \(f(x)\) 在点 \($x_0$\) 的某一**去心邻域**（即不包含 \($x_0$\) 本身，记为 \($\mathring{U}(x_0, \delta_0)$\)，其中 \($\delta_0 $> 0\)）内有定义。若存在常数 A，使得对任意给定的**任意小的正数 \($\varepsilon$\)**（通常称为 “误差限”），总存在正数 \($\delta$\)（通常与 \($\varepsilon$\) 相关，称为 “邻域半径”），当 \(0 < |x - $x_0$| <$ \delta$\) 时，恒有：\(|f(x) - A| < $\varepsilon$\)则称常数 A 为函数 \(f(x)\) 当 \($x \to x_0$\) 时的极限，记为：\($\lim_{x \to x_0} f(x) = A \quad \text{或} \quad f(x) \to A \quad (x \to x_0)$\)

### 收敛准则

**海涅定理**：

 $设函数 f(x) 在 x_0 的某去心邻域 U^\circ(x_0) 内有定义，则：\lim\limits_{x \to x_0} f(x) = A（A 为有限数或无穷大）**的充要条件是**：
对任意满足 \lim\limits_{n \to \infty} x_n = x_0\) 且 \(x_n \in U^\circ(x_0)\)（即 \(x_n \neq x_0\)）的数列 \(\{x_n\}\)，都有 \(\lim\limits_{n \to \infty} f(x_n) = A\)。$

**柯西准则：**

设函数 \(f(x)\) 在 \(x_0\) 的某去心邻域 \($U^\circ(x_0)$\) 内有定义，则：\($\lim\limits_{x \to x_0} f(x)$ 存在（且为有限数）**的充要条件是**：
对任意给定的 \($\varepsilon$ > 0\)，存在 \($\delta$ > 0\)，使得对任意 \($x_1, x_2 \in U^\circ(x_0)$\)，当 \($0 < |x_1 - x_0| < \delta$\) 且 \($0 < |x_2 - x_0| < \delta$\) 时，有 \(|f(x_1) - f($x_2)| < \varepsilon$\)。

**单调有界**：

- 若 \(f(x)\) 在 \((a, x_0)\) 内**单调递增且有上界**（或单调递减且有下界），则 \\$lim_{x \to x_0^-} f(x)$\) 存在；

- 若 \(f(x)\) 在 \((x_0, b)\) 内**单调递增且有下界**（或单调递减且有上界），则 \($\lim_{x \to x_0^+} f(x)$\) 存在。

**无穷小量的定义**：设函数\(f(x)\)在\(x $\to$ x_0\)（或\($x \to \infty$）的某去心邻域内有定义，若\($\lim_{x \to x_0} f(x) = 0$\)，则称\(f(x)\)是当\($x \to x_0$\)时的**无穷小量**，简称 “无穷小”。

## 函数的连续性

### 定义

**点连续**：

1. **\($f(x_0)$\) 存在**：函数在 \(x_0\) 处有确定的函数值（无 “无定义点”）；
2. **\($\lim\limits_{x \to x_0} f(x)$\) 存在**：函数在 \(x_0\) 处的极限存在（左极限 = 右极限，无 “跳跃间断”“无穷间断” 等）；
3. **\($\lim\limits_{x \to x_0}$ f(x) = f($x_0$)\)**：函数在 \(x_0\) 处的极限值等于该点的函数值（无 “可去间断”，即极限与函数值 “重合”）。

衍生出**左连续**和**右连续**的定义：

- **左连续**：若 \($\lim\limits_{x \to x_0^-} f(x) = f(x_0)$\)（左侧极限等于函数值），则称 \(f(x)\) 在 \($x_0$\) 处左连续；
- **右连续**：若 \($\lim\limits_{x \to x_0^+} f(x) = f(x_0)$\)（右侧极限等于函数值），则称 \(f(x)\) 在 \($x_0$\) 处右连续。

**区间连续**：

**开区间 \((a, b)\) 内连续**：若 \(f(x)\) 在开区间 \((a, b)\) 内的**每一点**都连续，则称 \(f(x)\) 在 \((a, b)\) 内连续；

**闭区间 \([a, b]\) 上连续**满足：

- 在开区间 \((a, b)\) 内连续；
- 在左端点 a 处**右连续**（\($\lim\limits_{x \to a^+} f(x) = f(a)$）；
- 在右端点 b 处**左连续**（\($\lim\limits_{x \to b^-} f(x) = f(b)$）； 则称 \(f(x)\) 在闭区间 \([a, b]\) 上连续。

**间断点**：

连续满足的条件：在$x_0$处有意义，左右极限存在且等于f($x_0$)。

**第一类间断点（左右极限都存在）**：

- **可去间断点**

左极限 = 右极限（即 \($\lim\limits_{x \to x_0} f(x) = A$\)，**极限存在**），但满足以下两种情况之一：

1. \(f(x_0)\) 无定义（函数在 \(x_0\) 处 “缺失”）；
2. \(f(x_0)\) 有定义，但 \(f(x_0) $\neq$ A\)（函数值与极限值不相等）。

- **跳跃间断点**

左极限 ≠ 右极限（即 \($\lim\limits_{x \to x_0^-} f(x) = A$\)，\($\lim\limits_{x \to x_0^+} f(x) = B$\)，且 \(A$ \neq$ B\)），无论 \(f(x_0)\) 是否有定义，均为跳跃间断点。

**第二类间断点**：

\($\lim\limits_{x \to x_0^-} f(x) 和 \lim\limits_{x \to x_0^+} f(x)$\) 中**至少有一个不存在**（包括极限为 \($\infty$\) 或振荡无固定趋势的情况）。

**一致连续：**

设f为定义在区间I上的函数，若对于任给的$\varepsilon$>0,存在$\delta$ =$\delta$($\varepsilon$)>0,使得对于任何$x^1$ $x^2$ $\in$ I,只要｜$x^1$ - $x^2$｜< $\delta$,就有｜f($x^1$)-f($x^2$) ｜<$\varepsilon$.则称f在区间I上一致连续。

### 性质

**点连续性质:**

- 四则运算
- 局部有界性
- 局部保号性
- 复合运算

**区间连续性质:**

- 最值定理（闭区间必有最大值与最小值）
- 介值性定理（设函数 \(f(x)\) 在**闭区间 \([a, b]\)** 上**连续**，且 \(f(a)$ \neq $f(b)\)。若 C 是介于 \(f(a)\) 和 \(f(b)\) 之间的任意一个实数（即 \($\min\{f(a), f(b)\} < C < \max\{f(a), f(b)\}$），则**至少存在一个点 \($\xi$$ \in$ (a, b)\)**，使得 \(f($\xi$) = C\)。当C为零时，为零点存在性定理。）
- 反函数连续性定理 若函数f在[a,b]上严格单调并连续，则反函数$f^-1$在其定义域上连续。
- 一致连续性定理(cantor) 若函数f在闭区间[a,b]上连续，则f在[a,b]上一致连续。
- 对**任意区间 I**（开区间、无限区间等皆可）上的函数 \(f(x)\)，一致连续的充要条件是： **对区间 I 内的任意两个序列 \(\{$x_n$\}\) 和 \(\{$y_n$\}\)，若 \($\lim\limits_{n \to \infty} (x_n - y_n) = 0$\)，则 \($\lim\limits_{n \to \infty} [f(x_n) - f(y_n)] = 0$\)**。
- 对于f(x)在开区间（a,b）连续，则f(x)在(a,b)上一致连续的充要条件：f在a的右极限，b的左极限存在且有限。

**初等函数连续性质：**

- 任何初等函数都属在其定义区间上的连续函数。

## 微分学

### 定义

**微分**：

设函数 \(y = f(x)\) 在点 \($x_0$\) 的某个邻域内有定义，若函数在 \($x_0$\) 处的增量 \($\Delta y = f(x_0 + \Delta x$) - f($x_0$)\) 可以表示为：\($\Delta y = A \cdot \Delta x + o(\Delta x$)\) 其中：

- A 是与 \($\Delta$ x\) 无关的常数（仅与函数 \(f(x)\) 和点 \($x_0$\) 有关）；
- \(o($\Delta$ x)\) 是当 \($\Delta x \to 0$\) 时，比 \(**$\Delta$** x\) 更高阶的无穷小量（即 \($\lim_{\Delta x \to 0} \frac{o(\Delta x)}{\Delta x} = 0$\)）；

则称函数 \(y = f(x)\) 在点 \(x_0\) 处**可微**，且把 \(A$ \cdot \Delta x$\) 称为函数 \(f(x)\) 在点 \(x_0\) 处的**微分**，记作 \(dy|_{x=x_0}\) 或 \(df(x_0)\)，即：\(dy|_{x=x_0} = A$ \cdot \Delta x$\)。dy称为线性主部。如果f在区间的每一点都可微，则在区间可微。

微分的本质是对函数增量 \($\Delta$ y\) 的 “简化近似”：

- **线性主部**（\(A $\cdot \Delta x$\)）：是 \($\Delta$ y\) 中与 \($\Delta$ x\) 成线性关系的部分，也是增量的 “主要贡献者”；
- **高阶无穷小**（\(o($\Delta$ x)\)）：当 \($\Delta$ x\) 很小时，这部分的影响可以忽略不计，是近似的 “误差项”。

**导数**：

设函数 \(y = f(x)\) 在点 \($x_0$\) 的某邻域内有定义，若极限：$\(\boxed{f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}}\) $存在，则称 \(f(x)\) 在 \($x_0$\) 处**可导**，该极限值即为 \(f(x)\) 在 \(x_0\) 处的导数，记为 \($f'(x_0)\)、\(y'|_{x=x_0}\) 或 \(\frac{dy}{dx}|_{x=x_0}$\)。

若该极限不存在，则称 \(f(x)\) 在 \($x_0$\) 处**不可导**.

**区间可导**：f在区间每一点都可导，当闭区间时，左（右）端点右（左）导存在。

**稳定点**：

对于函数 \(y = f(x)\)，若在点 \($x_0$\) 处满足以下两个条件：

1. 函数 \(f(x)\) 在 \($x_0$\) 处**可导**（即导数 \(f'($x_0$)\) 存在，排除了不可导点）；
2. 该点的导数等于 0，即 \($\boxed{f'(x_0) = 0}$\)；

则称点 \(($x_0$, f($x_0$))\) 是函数 \(f(x)\) 的**稳定点**（也常被称为 “驻点”，二者在微积分中含义完全等价）。

**高阶微分与导数**：

$设 \((n-1)\) 阶微分 \(d^{n-1} y = f^{(n-1)}(x) dx^{n-1}\)，则 n 阶微分是 \((n-1)\) 阶微分的微分：\(d^n y = d(d^{n-1} y) = d(f^{(n-1)}(x) dx^{n-1}) = f^{(n)}(x) dx \cdot dx^{n-1} + f^{(n-1)}(x) \cdot d(dx^{n-1})\)$
$因 \(dx^{n-1} = (dx)^{n-1}\) 是常数（\(d(dx^{n-1}) = 0\)），故化简为\(d^n y = f^{(n)}(x) \cdot dx^n\)$.

$设 \(f(x)\) 的 \((n-1)\) 阶导函数 \(f^{(n-1)}(x)\) 在 \(x_0 \in I_{n-1}\) 处可导（\(n \geq 2\)，\(I_{n-1} \subseteq I_{n-2} \subseteq \dots \subseteq I\)）$，则称 \(f(x)\) 在 \(x_0\) 处 **n 阶可导 **，记为$\(f^{(n)}(x_0) = \lim_{\Delta x \to 0} \frac{f^{(n-1)}(x_0 + \Delta x) - f^{(n-1)}(x_0)}{\Delta x}\)n 阶导函数记为 \(f^{(n)}(x)\)、\(y^{(n)}\) 或 \(\frac{d^n y}{dx^n}\)（注意：\(n \geq 3\) 时不用 \('''\)，而用上角标 \((n)\)）$

### 欧拉常数

**欧拉常数的定义**

欧拉常数通常用希腊字母 **γ**（gamma）表示，其最经典的定义源于**调和级数与自然对数的极限差**：

$对于正整数 n，调和级数的前 n 项和为 \(H_n = 1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n}\)。当 n 趋向于无穷大时，\(H_n\) 会趋向于无穷（调和级数发散），但如果用 \(H_n\) 减去自然对数 \(\ln n\)，其差值会收敛到一个固定的常数，这个常数就是欧拉常数：$

$\(\gamma = \lim_{n \to +\infty} \left( H_n - \ln n \right) = \lim_{n \to +\infty} \left( 1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n} - \ln n \right)\)$

这一极限由瑞士数学家欧拉（Leonhard Euler）在 1734 年首次研究并确认收敛，意大利数学家马斯凯罗尼（Lorenzo Mascheroni）后来对其进行了更精确的计算，因此欧拉常数也被称为 “欧拉 - 马斯凯罗尼常数”。

### 超越数

**定义**

设一个数为 $\alpha$，若不存在满足以下条件的多项式 \(P(x)\)：

1. 多项式系数均为**整数**（或等价地，均为有理数）；
2. 多项式次数 \(n $\geq$ 1\)（即非零常数多项式）；
3. \(P($\alpha$) = 0\)（即 \($\alpha$\) 是该多项式的根）。

则称 \($\alpha$\) 为**超越数**。

与之相对的概念是 “代数数”：能满足上述多项式方程的数（包括所有有理数、部分无理数如 \(\sqrt{2}\)、\(\sqrt[3]{5}\) 等）。

**性质**：

**超越数一定是无理数，但无理数不一定是超越数**
